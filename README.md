## ğŸš€ Getting Started

### Prerequisites
*   Node.js 18+
*   npm or yarn
*   **Google Gemini API Key** (Required for Search Agent)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/jay870423/LexiHub-Legal-AI.git
    cd lexihub
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory:
    ```env
    API_KEY=your_google_gemini_api_key_here
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```

## â˜ï¸ Deployment (Vercel)

This project is configured for one-click deployment on Vercel.

1.  Push your code to GitHub/GitLab.
2.  Import the project into Vercel.
3.  **Environment Variables**:
    *   Add `API_KEY` (Your Google Gemini API Key) in the Vercel Project Settings.
4.  **DeepSeek Proxy (Optional)**:
    *   The project includes `vercel.json` to proxy requests to `api.deepseek.com`.
    *   In the App Settings UI, set the DeepSeek Base URL to `/api/proxy/deepseek` to avoid CORS errors.
